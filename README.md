# Patio-Verde
Patio Verde es un espacio dedicado a la jardinería urbana, el cultivo doméstico y el cuidado de plantas en interiores y exteriores. Compartimos guías prácticas, consejos sobre iluminación, ventilación, sustratos y mantenimiento, así como contenidos relacionados con botánica, diseño verde y cultivo en espacios reducidos.
